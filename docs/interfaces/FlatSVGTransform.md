# Interface: FlatSVGTransform

2D affine transform — SVG `matrix(a b c d e f)`, equivalent to the matrix
`[[a c e][b d f][0 0 1]]` applied to a column vector `[x y 1]`. Identity
is `{ a: 1, b: 0, c: 0, d: 1, e: 0, f: 0 }`.
